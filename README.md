## Features

### 3D Visualization
- Highly realistic AI chip model
- PCB substrate with traces and capacitors
- Golden core DIE with circuit grid
- Integrated Heat Spreader (IHS)
- Heatsink with heat pipes
- Gold-plated pins

### Interactive Controls
- **Auto Rotate** - Automatic rotation display
- **Explode View** - Exploded view to reveal internal layers
- **Data Flow** - Particle animation for data visualization
- **Wireframe** - Toggle wireframe mode
- **Reset** - Reset camera and all settings

## Getting Started

Simply open `index.html` in a modern browser.

Or use a local server:

```bash
# Python
python -m http.server 8080

# Node.js
npx serve
```

Then visit `http://localhost:8080`
